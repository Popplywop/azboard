azboard

A fast, keyboard-driven terminal UI for Azure DevOps. Review pull requests, manage work items, and stay in your terminal.

Download latest release GitHub repository Full README

Quick install

Pick your platform and architecture, then copy the install command.

Linux / macOS

INSTALL_ARCH=amd64 curl -fsSL https://popplywop.github.io/azboard/install.sh | sh

Windows (PowerShell)

iwr https://popplywop.github.io/azboard/install.ps1 | iex

What you get

  • Pull request list/detail views with filtering, voting, comments, and merge actions
  • Code review workflow with file tree, diff viewer, iterations, and inline comments
  • Work item browsing, updates, comments, and PR linking
  • Config-driven setup for Azure DevOps org/project/repo selection
